This a checklist for contact

After succesfully generating your Celement with yeoman generator reraco, go through this list to check if everything fits for your website.
Component test incontact.test.js needs all prop parameters!
If not needed: Delete defaultAction from duck/actions.js and duck/actions.test.js
If not needed: Delete defaultOperation from duck/operations.js and duck/operations.test.js
If not needed: Delete defaultReducerFunction from duck/reducer.js and duck/reducer.test.js
Information: Usually defaultAction in duck/actions.js, defaultOperation in duck/operations.js, defaultReducerFunction in duck/reducer.js have the same name
